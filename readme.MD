# Requirement

Python 3, Node >= 14

# Installation

Install packages for frontend and backend

`yarn run install-packages`

Install packages Python

`cd cnn && pip install -r requirements.txt`

Install FFMPEG

`sudo apt install ffmpeg`

# Build / Compile

`yarn run build`

### Start server

`yarn start`
